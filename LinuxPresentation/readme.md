Bu sunum docker cli, docker file, docker compose cli ve docker compose file baliklarini içeren bastan sona öreneklendirilmis sunumdur.

Amaç developerlarin kendi ortamlarini olusturabilmeleri ve test yapabilmeleri için gerekli olan alyapiyi saglamaktir.

Sadece Linux sunucular üzerinden çalisma prensibini anlatmaktadir. Windows ile birlikte çalisma senaryolarini içermemektedir.
